## 基建清单（低成本版）

### (1) 阵地：公网跳板机 + 旧笔记本复活

很多人为了跑AI服务，花大价钱买高配云服务器，这是最大的误区。  
云服务器寸土寸金，但你家里那台吃灰的旧电脑，其实是座金矿。

**云端跳板（Gateway）**：买一台最便宜的**公网IP云主机**（一年约200元，5M带宽）。它的配置不重要（1核2G都行），重要的是它有一个**公网IP**。

**家庭算力（Worker）**：翻出你家里那台 **7、8年前的旧笔记本**。 哪怕是7年前的 8核CPU、8G内存、256G固态硬盘，它的能力也**吊打**你每年几百块买的“2核4G”云服务器。配合家里139块**千兆下行**的光纤网络，这就相当于你在家里部署了一个私有的小型数据中心。

### (2) 连接：WireGuard + IPv6

有了两台机器三个手机，怎么连起来？必须掌握的**核心技能是WireGuard**。

你必须学会搭建 WireGuard 加密链路。它能把你的手机、云主机、家里的旧笔记本，全部虚拟在同一个局域网里。

你在单位摸鱼时，手机可以直接控制家里笔记本上的Docker容器；你的AI Agent跑在家里笔记本上，却可以通过云主机的公网IP对外提供服务。

**进阶玩法：IPv6**。如果你家里的宽带有 **公网IPv6地址**，且上传带宽在 **20Mbps以上**，恭喜你，你甚至不需要云主机转发。你可以随时随地通过 IPv6 直接连接回家，流畅地观看家里NAS里的 **4K电影**，或者远程操作高性能桌面。

_不会搭？_ 没关系，把“如何用Docker部署WireGuard”这句话发给AI，它会一步步教你，连配置文件都给你写好。

### (3) 土壤：Linux + Docker是AI刚需

**Docker**是必需的工具，但你得会配置加速器，你直接拖，搞不下来的。

不管你在Github上看到什么牛逼的开源项目（比如DeepSeek的本地版、各种RAG知识库），只要有Docker，通常一行命令`docker run`就能跑起来。它让你免去了配置环境的痛苦。

### (4) 燃料：便宜好用的API Key

不要只盯着OpenAI。你需要寻找那些**便宜、垂直、好用**的 API。

**搜索能力**：**Brave Search API**。极其便宜，且保护隐私，让你的AI Agent拥有联网搜索的能力。

**阅读能力**：**Jina.ai**。把任何网页瞬间变成适合AI阅读的 Markdown 格式。

**爬虫能力**：**Firecrawl**。专门用来抓取复杂的动态网页数据。

**翻译能力**：**DeepL API**。比谷歌翻译更地道，每月有免费额度。

### (5) 手脚：命令行Code工具

这是目前最被低估的神器，openclaw大家不知道用过没有，或者各种命令行AI大家用过没有？

**Gemini CLI / Claude Code / OpenCode**：这些工具允许你在终端里直接用自然语言下令。

你对AI工具说，在我的笔记本上部署一个服务，每天早上调用 Brave Search 搜索美股新闻，用 DeepL 翻译，然后推送到我的手机上。香不香？

或者，跟AI工具说，你看看哪些公司有财务造假风险，特别是理财收益不到1%的公司，详细分析一下财报和他们的舆情。这不就是避坑吗？？
